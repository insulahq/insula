# Insula brand kit

Canonical logo and brand assets. **This directory is the single source of truth** —
the documentation site and both panels reference copies of these files; update
here first.

## Logo

The mark is a **hexagonal node cluster** (a Kubernetes node) with a hub-and-spoke
mesh and one lit node — isolated tenants connected by one platform. The wordmark
is **Sora**, converted to vector outlines so the logo carries **no font
dependency**.

| File | Use |
|---|---|
| `insula-mark.svg` | The mark on its own (favicons, app icon, square contexts) |
| `insula-lockup.svg` | Mark + "Insula" wordmark — **primary logo** on light backgrounds |
| `insula-lockup-dark.svg` | Lockup for dark backgrounds (cyan wordmark) |
| `insula-mono.svg` | One-colour knockout (`fill: currentColor`) — stamps, print, single-colour UI |
| `insula-mark-white.svg` | Reversed mark for solid/dark backgrounds |
| `favicon.svg` | Vector favicon (= the mark) |
| `favicon-16/32/48/180/512.png`, `apple-touch-icon.png`, `favicon.ico` | Raster favicons |

## Colours

| Token | Hex | Use |
|---|---|---|
| Teal (base) | `#0D9488` | Hexagon body, primary brand colour |
| Mint (mesh) | `#99F6E4` | Spokes & unlit nodes |
| Orange (accent) | `#FB923C` | The lit node; call-to-action accent |
| Wordmark on light | `#0F766E` | "Insula" text on light backgrounds |
| Wordmark on dark | `#5EEAD4` | "Insula" text on dark backgrounds |

## Typography

- **Logo wordmark:** Sora 700 (outlined in the SVG — not loaded at runtime).
- **Suggested UI/heading font:** Sora, if/when the panels adopt a brand font.
  The documentation site stays on system fonts (`font: false`) by design.

## Clear space & minimum size

Keep clear space of at least half the mark's height around the logo. Minimum
legible size: 16 px for the mark (favicon), ~96 px wide for the lockup.

## Regenerating

Assets are generated by the script captured in the brand PR (hand-authored SVG +
`cairosvg`/`fonttools`). To change the mark or palette, edit the source SVGs here
and re-export the raster favicons.
